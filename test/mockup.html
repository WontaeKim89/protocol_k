weather.short_forecast T2
기상청 · 환경기상 · 인증: serviceKey (query) · 인기 #2

단기 예보 조회

기상청 격자좌표(nx, ny) 기반의 1~3시간 단위 단기 예보를 반환합니다. base_date / base_time 은 3시간 단위로 정해진 발표 시각만 사용 가능합니다.

Short-term weather forecast by KMA grid coordinates. Returns 1~3 hour interval forecasts.
⚠ 함정 주의

nx·ny 는 위경도가 아닙니다. 기상청 자체 격자좌표입니다. 위경도 변환은 별도 Resolver tool(addr_to_grid, Phase 2) 사용을 권장합니다. base_time 은 0200, 0500, 0800, 1100, 1400, 1700, 2000, 2300 중 하나여야 합니다.
